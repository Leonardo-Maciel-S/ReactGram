# ReactGram

- Projetinho FullStack em React + Node com Express para aplicar conhecimentos mais aprofundados como:
## Front End
- Redux com toolkit
- Consumo de api
- React router
- Controle de acesso com autenticação
- Hooks
## Back end
- Criação de api
- Express
- Rotas
- jwtSecret
- Autenticação
- Middlewares
- Controllers
- Models (MongoDB com mongoose)
